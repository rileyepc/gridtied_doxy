◆ APP_EEPROM_SECTIONS_DEFAULTS

#define APP_EEPROM_SECTIONS_DEFAULTS
Note
Section revision must be updated under the following conditions:
  1. Any addition, subtraction, or size change to a meta parameter section (meta ruins everything)
  2. Removing one or more parameters from a section
  3. Changing the size of an existing parameter in a section
  4. Re-ordering parameters within a section
  5. Adding a parameter anywhere other than to the end of a section

In the future this should/will be managed and generated automatically by PM, preferably with memory storage managed by an OS file system running on a SoC.
