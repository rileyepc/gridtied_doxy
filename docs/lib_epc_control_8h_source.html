178 * Select a "switched" model intent for accurate modelling of switching-frequency sideband harmonics at

181 * simulation speed. */

238/* Input/Output Port Specification:
239 * Input and output ports are the process data interfaces to the EPC Control in the SIL environment.
240 * Ports may be scalar or vectorial (arrays). Vectorial ports are used to group (roughly) like data.
241 * In some ways and cases, the groupings may represent similar groupings as exposed in the real control
242 * interfaces (Multiplexed CAN messages or Modbus register groups), but not necessarily equivalent in
243 * the interest of offering groupings more relevant to the SIL Library use case. Ports are exposed by
244 * reference (pointer). Pointers must be configured to valid memory locations by the host simulation
245 * environment by the time SIL_initializeLibrary() method is called. Vectorial ports are required to
246 * occupy contiguous memory locations. Please refer to SIL_INPUTS_DEFAULTS() and SIL_OUTPUTS_DEFAULTS()
247 * macros above for reference to the expected array sizes of each port. Typically the simulation
248 * environment will iterate over the array elements of inpPort and outPort respectively to allocate
249 * memory and assign pointers for use by the SIL Library. */
250typedef struct SIL_PortSpec_s {
251 double *portPointer;
252 int portWidth;
253} SIL_sPortSpec_t;

424/* Remarks:
425 * † Typical assignment for AC-only Controls Variants. Topology+Control Variant dependent in general.
426 *
427 * ‡ Presently, Digital Inputs and Outputs may be left disconnected as they are faked out in Library.
428 *
429 * ¹ "Local" voltage measured on network/grid-side of the main contactor.
430 *
431 * ² "Remote" voltage measured typically at PCC for power control at that point, or on a protected bus
432 * for UPS operation, synchronous transfer, etc. when so configured/desired.
433 *
434 * ³ For the switched model intent, the "Gate" outputs are the instantaneous switching states of the
435 * non-complimentary switching devices comprising the power stage. Complementary-device gate signal
436 * expansion, active logic level selection, and optional cross-conduction dead-band time insertion
437 * must be treated externally. For the averaged model intent, the "Gate" outputs are normalized
438 * values on the bounded real interval [-1, 1].
439 *
440 * ⁴ DG (Distributed Generation) is the default grid-interactive Controls Variant.
441 *
442 * ⁵ All ports must be assigned valid pointers (in contiguous memory for vectorial ports), whether the
443 * port is used/desired or not. The user may declare a dummy array for assigning pointers to unused
444 * input or output ports.

459/********************************************************************************************************
460 Function : SIL_initializeLibrary
461 Full Name : Grid-Tied::EpcSilLibrary.initializeLibrary
462 Visibility : Public
463
464 In : tStep - requested simulation timestep [s],
465 solverType - enumerated selection indicating solver capabilities [-],
466 mdlIntent - enumerated selection indicating model intent [-],
467 bridgeType - enumerated selection indicating EPC Power Bridge type [-],
468 pmvsPath - path to PMVS file.
469 InOut : None
470 Out : None
471 Returns : True on success; False otherwise.
472 Description : Configures the SIL library according to the dynamic simulation environment's capabilities
473 and modeling objectives. This method is to be called once prior to progression into the
474 simulation loop phase. For fixed-rate solvers, tStep shall indicate the discretization
475 time step. For solvers capable of supporting variable rates, tStep is ignored.
476 NB! The selected fixed sample rate must be an integral submultiple of all sample times
477 required by the library: ts, 200μs, 1ms, 10ms, 100ms, and 1000ms. Presently for fixed-rate
478 solver selection, the "ts" (realtime) control task is fixed at 150μs for a switching
479 frequency of 3333Hz, but this will be expanded in the future to support a configurable
480 switching frequency subject to validation given the sample step time. For now, the sample
481 rate should be the greatest common divisor of these internal sample rates, or 50μs.
482*********************************************************************************************************/
483bool SIL_initializeLibrary(double tStep, SIL_eSolverType_t solverType, SIL_eModelIntent_t mdlIntent,
484 char *pmvsPath);
