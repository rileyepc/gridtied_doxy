\begin{DoxyNote}{Note}
Section revision must be updated under the following conditions\+:
\begin{DoxyEnumerate}
\item Any addition, subtraction, or size change to a meta parameter section (meta ruins everything)
\item Removing one or more parameters from a section
\item Changing the size of an existing parameter in a section
\item Re-\/ordering parameters within a section
\item Adding a parameter anywhere other than to the end of a section
\end{DoxyEnumerate}
\end{DoxyNote}
In the future this should/will be managed and generated automatically by PM, preferably with memory storage managed by an OS file system running on a SoC.
